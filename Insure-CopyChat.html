📋 사용 순서

아래 순서대로 📋 복사 버튼을 눌러 ChatGPT에 붙여넣으세요.

1단계  시작 설정 (처음 한 번만)

AI에게 오늘 만들 콘텐츠의 방향을 미리 알려주는 단계예요. 처음 한 번만 하면 됩니다.
복사해서 ChatGPT에 붙여넣으세요.

오늘 날짜를 기준으로 가족의 건강과 노후를 고민하는 40~70대 타겟으로 보험 마케팅 콘텐츠를 만들 거야. 내가 첨부하는 상품 자료를 반드시 바탕으로 작성해줘. 아래 세 가지 작업을 순서대로 진행할 거야: 1. 카드뉴스 5장 기획 문구: 1장(관심 끌기) / 2장(문제 제기) / 3장(해결책·상품 장점) / 4장(차별점) / 5장(문의 유도) 2. 인스타그램 릴스 캡션: - 본문 3문단: 1문단(공감) → 2문단(핵심 내용) → 3문단(행동 유도) - 해시태그 15개: 40~70대·보험·건강·노후 관련 키워드로 작성 - 마지막 줄: "카카오톡으로 편하게 문의하세요 ☝️ 프로필 링크 클릭!" 유도 멘트 - 이모지 적절히 활용 3. 클링 AI 3.0 멀티샷 영상 프롬프트 (요청 시): 총 7초 / 3~4샷 구성 / 아래 형식으로 한국어 작성 / 자막 없음 [샷 N | 0:00~0:00] 장면: (배경·인물·조명 묘사) 카메라: (무빙 방식) ※ 립싱크 없음 / 배경음악·효과음 생성 / 자막 없음
2단계  상품 자료 올리기

📎 파일이나 내용을 ChatGPT에 올려주세요

1단계에서 이미 자료를 바탕으로 작업하겠다고 설정했기 때문에,
따로 명령어를 보낼 필요가 없어요.

  • ChatGPT 입력창 왼쪽 📎 아이콘으로 파일 첨부
  • 또는 상품 내용을 직접 입력창에 붙여넣기
  • 올리고 나면 AI가 자동으로 분석하여 반영합니다
3단계  카드뉴스 이미지 만들기

카드뉴스 5장을 한 장씩 순서대로 만드는 단계예요. 한 장 완성 후 다음 번호로 넘어가세요.
하나씩 복사해서 ChatGPT에 붙여넣으세요.

1번 이미지

방금 짜준 기획안 1장 내용을 바탕으로 4:5 비율 이미지를 만들어줘. 글자는 한글로 뚜렷하게. 이미지 오른쪽 상단에 캐러셀 번호 '1 / 5'를 작게 넣어줘.

2번 이미지

기획안 2장 내용을 바탕으로 4:5 비율 이미지를 만들어줘. 이미지 오른쪽 상단에 캐러셀 번호 '2 / 5'를 작게 넣어줘.

3번 이미지

기획안 3장 내용을 바탕으로 4:5 비율 이미지를 만들어줘. 이미지 오른쪽 상단에 캐러셀 번호 '3 / 5'를 작게 넣어줘.

4번 이미지

기획안 4장 내용을 바탕으로 4:5 비율 이미지를 만들어줘. 이미지 오른쪽 상단에 캐러셀 번호 '4 / 5'를 작게 넣어줘.

5번 이미지

기획안 5장 내용을 바탕으로 4:5 비율 이미지를 만들어줘. 이미지 오른쪽 상단에 캐러셀 번호 '5 / 5'를 작게 넣어줘.
4단계  인스타그램 글 & 해시태그

만든 이미지에 맞는 게시물 글과 해시태그를 자동으로 작성해줘요. 이미지 5장 완성 후 진행하세요.
복사해서 ChatGPT에 붙여넣으세요.

방금 만든 카드뉴스 5장을 인스타그램에 올릴 거야. 게시물 글을 작성해줘. - 본문: 3문단 구성 (1문단: 공감 / 2문단: 핵심 내용 / 3문단: 행동 유도) - 마지막 줄: "카카오톡으로 편하게 문의하세요 ☝️ 프로필 링크 클릭!" 유도 멘트 - 해시태그 15개: 40~70대, 보험, 건강, 노후 관련으로 작성 - 이모지를 적절히 활용해서 읽기 편하게
5단계  영상 만들기

클링 AI에서 쓸 세로 이미지와 영상 프롬프트를 준비하는 단계예요. 아래 순서대로 진행하세요.
복사해서 ChatGPT에 붙여넣으세요.

① 영상 기획하기

내가 첨부한 자료와 지금까지 만든 카드뉴스·게시물 내용을 바탕으로 7초 릴스 영상의 스토리 방향과 핵심 메시지를 먼저 기획해 줘.

② 영상용 세로 이미지 만들기

방금 기획한 내용대로, 앞서 만든 카드뉴스 게시물에 등장한 캐릭터를 참고해서 9:16 세로 비율의 캐릭터 이미지 1장을 만들어 줘. 정면을 응시하며 시청자의 시선을 멈추게 할 강렬한 눈빛과 표정으로 표현해 줘.

③ 영상 내용 짜기

방금 기획한 내용대로, 방금 만든 세로 이미지를 활용한 클링 AI 3.0 자동 멀티샷(Auto Multi-Shot) 영상 프롬프트를 작성해 줘. - 클링 AI 메인 프롬프트 입력창에 바로 붙여넣을 수 있는 단일 텍스트 블록으로 작성 (2,500자 이내) - 완성된 프롬프트는 설명 없이 프롬프트 본문만 코드 블록(```)으로 감싸서 출력해 줘 (바로 복사·붙여넣기 가능하도록) - 첫 1~3초: 40~70대가 스크롤을 멈출 만한 강렬한 후킹 대사나 장면으로 시작 - 두려움·공감·놀라움 중 하나를 자극하는 감정 흐름으로 나레이션 구성 - 문제 제기 → 해결책 → 행동 유도 순서로 전개 - 나레이션 타이밍: 0~6초 구간에만 배치, 마지막 1초(6~7초)는 나레이션 없이 여운 장면으로 마무리 - 나레이션 대사는 한글 기준 전체 15자 이내로 엄격히 제한 (TTS 자동 생성 시 잘리지 않도록) - 짧은 문구 1~2개로만 구성, 긴 문장·복합문·어려운 표현 절대 금지 - 문장과 문장 사이는 쉼표나 점으로 끊어 TTS가 자연스럽게 쉬어갈 수 있도록 설계 - 마지막 여운 장면(6~7초): 나레이션 없이 브랜드 로고가 화면 중앙에 자연스럽게 페이드인으로 등장하며 마무리 - 총 7초 / 립싱크 없음 / 배경음악·효과음 생성 / 한글 자막 없음 (AI 자막 오류 방지)
